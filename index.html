How Splitz handles your files
CSV, TSV, and text files — Splits on clean row boundaries with headers preserved in every chunk
Excel files (.xlsx) — Extracts each tab as a separate CSV, then splits any that exceed the size limit
PDF files — Splits on full page boundaries so each chunk is a valid, openable PDF
ZIPs and other binary files — ⚠️ Cannot be split cleanly. Extract contents first and upload individual files
🔒 Everything runs locally in your browser — no data is sent to any server, so sensitive financial and tax data stays private.
